# Stuart Crowley LLM Voice Pack

This folder is a practical instruction set for any LLM writing as Stuart Crowley or supporting NEXTPredict.

## Start here

1. Paste `01_MASTER_SYSTEM_PROMPT.txt` into the model's system or custom-instructions field.
2. Give the model the task and any source intel.
3. Use the relevant task prompt from `08_TASK_PROMPTS.md` when extra control is needed.
4. Run the output against `10_FINAL_CHECKLIST.md` before publishing.

## Files

- `01_MASTER_SYSTEM_PROMPT.txt`: Full paste-ready instruction set.
- `02_COMPACT_SYSTEM_PROMPT.txt`: Shorter version for models with limited instruction space.
- `03_STUART_VOICE_GUIDE.md`: Tone, rhythm, preferred constructions and banned habits.
- `04_PLATFORM_PLAYBOOK.md`: LinkedIn, Prediction Markets Forum, X, comments, DMs and email.
- `05_NEXTPREDICT_POSITIONING.md`: Event positioning, commercial intent and relationship sensitivities.
- `06_INTEL_TRIAGE_AND_VERIFICATION.md`: What matters, what to ignore and how to handle unverified claims.
- `07_APPROVED_EXAMPLES.md`: Voice references and example patterns.
- `08_TASK_PROMPTS.md`: Reusable prompts for posts, comments, DMs, outreach and intel triage.
- `09_NEXTPREDICT_FACT_SHEET.md`: Stable event facts and approved language.
- `10_FINAL_CHECKLIST.md`: Pre-publication quality control.
- `llm_profile.json`: Machine-readable profile.

## Default operating rule

Write as Stuart unless the task explicitly says to write as NEXTPredict or NEXT.io.

For every social-post topic, provide three platform-specific versions:

1. Stuart's personal LinkedIn
2. Prediction Markets Forum LinkedIn group
3. X/Twitter

The three versions must be adapted to the platform rather than copied verbatim.
